# FESDFA Security Assessment

## Project Overview

FESDFA (Fédération des Serviteurs de Dieu Francophones en Amérique) is a nonprofit organization serving faith-based leaders and ministries.

The purpose of this assessment is to identify risks, evaluate existing controls, and recommend security improvements for the organization's membership management platform.

## Security Objectives

- Protect member information
- Ensure data confidentiality
- Maintain data integrity
- Improve availability of services
- Reduce cybersecurity risks
- Establish security best practices

## Scope

- Website
- Membership database
- QR verification system
- Registration forms
- Administrative accounts
