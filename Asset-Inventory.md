# FESDFA Asset Inventory

## Overview

An asset inventory identifies and categorizes the information systems, data, and resources that must be protected.

## Critical Assets

| Asset | Description | Importance |
|---------|-------------|------------|
| Website | Public-facing FESDFA website | High |
| Membership Database | Stores member information | Critical |
| QR Verification System | Verifies membership credentials | High |
| Admin Accounts | Manage members and content | Critical |
| Member Photos | Profile and identification images | Medium |
| Registration Forms | Collect member information | High |
| Netlify Hosting | Hosts website services | High |
| Email Accounts | Official communications | High |

## Data Assets

| Data Type | Classification |
|------------|--------------|
| Member Names | Public |
| Ministry Information | Public |
| Membership ID | Internal |
| Date of Birth | Confidential |
| Phone Number | Confidential |
| Email Address | Confidential |
| Administrative Notes | Confidential |

## Security Objective

Protect assets through appropriate security controls that ensure:

- Confidentiality
- Integrity
- Availability
